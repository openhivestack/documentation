This quick start guide will help you implement your first H.I.V.E. Protocol agent with minimal setup. We'll create a
simple agent that can provide a text processing capability and interact with other agents.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ installed
- Basic understanding of JavaScript/TypeScript
- Terminal or command prompt access

## Step 1: Set Up Your Project

Create a new project directory and initialize it:

```bash
# Create project directory
mkdir my-hive-agent
cd my-hive-agent

# Initialize npm project
npm init -y

# Install dependencies
npm install @hive-protocol/sdk express cors
npm install --save-dev typescript ts-node @types/node @types/express @types/cors

# Initialize TypeScript
npx tsc --init
```

## Step 2: Create Agent Configuration

Create a configuration file `config.ts` for your agent:

```typescript
// config.ts
export const agentConfig = {
  id: 'text-processor-agent',
  name: 'Text Processing Agent',
  description: 'Provides basic text processing capabilities',
  version: '1.0.0',
  endpoint: 'http://localhost:3000',
  // Registry to register with (optional)
  registryUrl: 'https://registry.example.com'
};
```

## Step 3: Implement the Agent

Create your main agent file `agent.ts`:

```typescript
// agent.ts
import { HiveAgent, Capability, KeyPair } from '@hive-protocol/sdk';
import express from 'express';
import cors from 'cors';
import { agentConfig } from './config';

async function main() {
  // Create Express app
  const app = express();
  app.use(cors());
  app.use(express.json());

  // Generate or load keys
  const keyPair = await KeyPair.generate('Ed25519');
  console.log(`Agent public key: ${keyPair.publicKey}`);

  // Create the agent
  const agent = new HiveAgent({
    id: agentConfig.id,
    name: agentConfig.name,
    description: agentConfig.description,
    version: agentConfig.version,
    keyPair,
    endpoint: agentConfig.endpoint
  });

  // Define text processing capability
  const textProcessingCapability = new Capability({
    id: 'text-processing',
    name: 'Text Processing',
    description: 'Performs basic text processing operations',
    parameters: {
      operation: {
        type: 'string',
        required: true,
        enum: ['uppercase', 'lowercase', 'reverse', 'count']
      },
      text: {
        type: 'string',
        required: true,
        maxLength: 10000
      }
    },
    handler: async (params) => {
      const { operation, text } = params;

      switch (operation) {
        case 'uppercase':
          return { result: text.toUpperCase() };
        case 'lowercase':
          return { result: text.toLowerCase() };
        case 'reverse':
          return { result: text.split('').reverse().join('') };
        case 'count':
          return { result: text.length };
        default:
          throw new Error(`Unknown operation: ${operation}`);
      }
    }
  });

  // Register the capability
  agent.registerCapability(textProcessingCapability);

  // Set up HTTP routes
  app.use('/api', agent.createExpressRouter());

  // Add a simple status endpoint
  app.get('/status', (req, res) => {
    res.json({
      status: 'ok',
      agent: agentConfig.id,
      version: agentConfig.version
    });
  });

  // Start the server
  const port = 3000;
  app.listen(port, () => {
    console.log(`Agent server running at http://localhost:${port}`);
  });

  // Register with discovery service if configured
  if (agentConfig.registryUrl) {
    try {
      await agent.registerWithDiscoveryService(agentConfig.registryUrl);
      console.log(`Registered with discovery service at ${agentConfig.registryUrl}`);
    } catch (error) {
      console.error('Failed to register with discovery service:', error);
    }
  }
}

// Run the agent
main().catch(console.error);
```

## Step 4: Create a Task Client

Create a simple client `client.ts` to interact with your agent:

```typescript
// client.ts
import { HiveClient, KeyPair } from '@hive-protocol/sdk';

async function main() {
  // Generate client keys
  const keyPair = await KeyPair.generate('Ed25519');

  // Create a client
  const client = new HiveClient({
    id: 'simple-client',
    keyPair
  });

  // Connect to your agent
  const agentEndpoint = 'http://localhost:3000/api';

  try {
    // Request a capability
    console.log('Sending task request...');
    const result = await client.requestTask({
      endpoint: agentEndpoint,
      capability: 'text-processing',
      parameters: {
        operation: 'uppercase',
        text: 'Hello from the H.I.V.E. Protocol!'
      }
    });

    console.log('Task completed successfully!');
    console.log('Result:', result);
  } catch (error) {
    console.error('Error executing task:', error);
  }
}

// Run the client
main().catch(console.error);
```

## Step 5: Set Up Your Package Scripts

Update your `package.json` to include these scripts:

```json
{
  "scripts": {
    "start": "ts-node agent.ts",
    "client": "ts-node client.ts"
  }
}
```

## Step 6: Run Your Agent

Start your agent in one terminal:

```bash
npm start
```

You should see output like:

```
Agent public key: <your public key>
Agent server running at http://localhost:3000
```

## Step 7: Test with the Client

In another terminal, run the client to test your agent:

```bash
npm run client
```

You should see output like:

```
Sending task request...
Task completed successfully!
Result: { result: 'HELLO FROM THE H.I.V.E. PROTOCOL!' }
```

## Troubleshooting

### Common Issues

- **Connection Refused**: Make sure your agent is running before testing with the client
- **Signature Verification Failed**: Check that you're using the correct key pairs
- **Capability Not Found**: Verify that the capability ID matches what's registered
- **Invalid Parameters**: Ensure parameters match the capability's requirements

<Callout type="warning">
  In a production environment, you should never hardcode or commit private keys. Use environment variables or
    secure key management solutions instead.
</Callout>
